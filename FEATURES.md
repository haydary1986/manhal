# 🎓 المركز الأكاديمي — مواصفات المميزات (Feature Spec)

بوت تلكرام خدمي موحّد يخدم **الباحثين والطلبة** ويكون **مرجعاً** يومياً، مع قناة إعلانات أكاديمية (مؤتمرات/منح) تبني الجمهور. خدمة عامة مستقلة (ليست باسم جهة رسمية).

---

## 🎯 الجمهور
- **الباحثون / التدريسيون** (أولوية): نشر، Quartile، اقتباسات، ترقيات.
- **الطلبة** (بكالوريوس + دراسات عليا): اقتباسات، تلخيص، إحصاء، دراسة.

## 🧱 المعمارية (هجين — قرار مجلس الخبراء بالإجماع)
**نواة Go واحدة (Core API)** تخدم قناتين فوق نفس المنطق:
- **بوت تلكرام (thin adapter):** الدخول + النمو + بوابة الاشتراك + التنبيهات + الميزات الخفيفة + الدفع.
- **منصة ويب (adapter ثانٍ):** الميزات الثقيلة بصرياً + لوحة admin + بوابة B2B المؤسسية.
- **الجسر:** البوت يرسل **رابط دخول موقّع لمرة واحدة (deep-link)** يفتح الويب مسجَّلاً تلقائياً (صفر تسجيل ثانٍ) بعد التحقق من الاشتراك بالقناة.
- **القاعدة:** مخرج سطر/سطرين → تلكرام؛ مخرج مستند أو لوحة تفاعلية → ويب.
- **التسلسل:** تلكرام أولاً (مراحل ٠–١)، ثم طبقة الويب للثقيل وB2B (مرحلة ٢+).
- **التقنية:** Go + `go-telegram/bot` + PostgreSQL (`pgx/v5`) + pgvector + **DeepSeek** (مزوّد قابل للتبديل؛ بلا قراءة PDF أصلية/سياق ~64K/بلا embeddings → نستخرج نص الـ PDF بـ Go، ونضيف مصدر embeddings بمرحلة RAG). Local Bot API server يرفع حد الملفات لـ ~2GB عند الحاجة.
- بنية نظيفة: `core (المنطق) / store / domain / scholar / cite / ai / announce / billing` + adapters: `bot / web`.

## ⚖️ المبادئ
- **Open Access قانوني فقط** — لا Sci-Hub/Nexus/LibGen.
- **بيانات استرشادية (advisory)** — لا ادعاءات قطعية (المجلات المفترسة، الفجوة البحثية).
- **عربي بالكامل** بواجهة أزرار.
- **إعادة الصياغة وتحسين الكتابة = مسموح** (تحرير شريف لنصّك). الممنوع: **عدّاد/محسّن لكسر كاشفات الـ AI** أو تسويقها كـ"إخفاء نص AI". فحص التشابه **استرشادي** وليس بديل Turnitin. لا Google Scholar (بلا API).

## 💰 نموذج الربح
- **Freemium**: مجاني سخي يجذب + **Premium** يفتح "السحر".
- **الدفع (طبقة billing مجرّدة، التحصيل OFF عند الإطلاق):** بما أنه **لا توجد شركة مرخّصة حالياً** (وبوابات ZainCash/FastPay التاجرية تتطلب شركة) → عند التفعيل: **Telegram Stars** كقناة آلية (بلا شركة، اشتراك متجدد، عمولة ~٣٠٪) + **أكواد/ZainCash شخصي يدوي** كبديل رخيص للعراقيين. **لاحقاً بعد تسجيل شركة:** تنضاف بوابة **ZainCash/FastPay** التاجرية (عمولة <٣٪) كقناة أساسية. B2B عبر تحويل بنكي + فاتورة على الويب.
- **طبقتان** (مرنة، قابلة للتعديل من اللوحة):
  - 🎓 طالب: ~**150 ⭐**/شهر (~$3)
  - 🔬 باحث: ~**400 ⭐**/شهر (~$8)
  - 📅 سنوي بخصم (~شهرين مجاناً).
- **مفاتيح التحكم (Feature Flags):** كل ميزة لها `{enabled, tier, daily_limit}`؛ مفتاح عام `monetization_enabled`. **تُبنى كل المميزات المدفوعة وتبقى نائمة، وتُفعّل من الأدمن لاحقاً** بدون إعادة نشر.
- **المنح اليدوي:** الأدمن يمنح Premium مجاناً (دائم/مؤقت/جماعي) + **أكواد هدية**.
- **نظام نقاط/كريدت:** للعمليات الثقيلة (وكلاء البحث، الإحصاء، الدفعات) — بديل/مكمّل للاشتراك.
- **ترخيص مؤسسي (B2B):** باقة للكليات/الأقسام — دخل مؤسسي حقيقي بالعراق (أفضل من Enterprise فردي).

## 🧩 كتالوج المميزات

### ⚙️ النظام (بنية)
- واجهة عربية + أزرار تفاعلية + جلسات wizard.
- بوابة الاشتراك الإجباري (force-subscribe).
- تسجيل المستخدم + التخصص/الكلية.
- سقوف استخدام يومية للـ AI (حتى بالوضع المجاني — لضبط الكلفة).
- معالجة أخطاء برسائل عربية + حدود حجم الملفات.

### 📢 المجموعة A — الإعلانات والتنبيهات (مجاني — يبني الجمهور)
1. إعلانات المؤتمرات و CFP + فلترة بالتخصص.
2. إعلانات المنح والزمالات والوظائف.
3. تذكير المواعيد النهائية.
4. منبّه النشر (أوراق جديدة لباحث/موضوع).
5. منبّه الاستشهاد (اقتباس جديد لبحثك).
6. ترندات المجال (الأكثر اقتباساً مؤخراً).
7. متابعة/إلغاء مواضيع وباحثين.

### 🔎 المجموعة B — نواة المرجع (مجاني)
8. بحث الأوراق بالعنوان/المؤلف/الكلمة (Semantic Scholar + OpenAlex).
9. جلب بيانات ورقة بالـ DOI (Crossref).
10. ⭐ مولّد الاقتباس التلقائي — DOI/ISBN → APA/MLA/Chicago/Harvard/IEEE/Vancouver + BibTeX.
11. باحث النص الكامل القانوني — Unpaywall + arXiv + CORE + OpenAlex + «اطلب من المؤلف».
12. فاحص تصنيف المجلة (Quartile) — Scimago + DOAJ.
13. كاشف المجلات المفترسة (advisory).
14. ملف الباحث / h-index (OpenAlex).

### 🧠 المجموعة C — المساعد الذكي (DeepSeek، بسقف)
15. مساعد الكتابة الأكاديمية (صياغة/تدقيق ع+إ).
16. تلخيص ورقة/PDF.
17. باني خطة البحث (Proposal Architect — wizard).
18. مولّد الاختبارات/الكويزات من الملاحظات.
19. اقتراح عناوين وأفكار بحثية.
20. الترجمة الأكاديمية ع↔إ.

### 📚 المجموعة D — المتقدّم
21. مدير المراجع الشخصي الذكي (تصنيف + كلمات مفتاحية تلقائية).
22. استدعاء المكتبة + بحث دلالي (pgvector).
23. تصدير المكتبة BibTeX / RIS.
24. محادثة الـ PDF (RAG).
25. كاشف الفجوة البحثية (advisory).

## 🌟 مميزات Premium القاتلة (محرّكات الاشتراك)

### للباحثين
- **P1 🧭 مرشّح المجلات «وين أنشر؟»** — ملخص → أنسب المجلات (مطابقة + Quartile + Scopus + APC + تحذير مفترسة). *يحل وجع #١.*
- **P2 📖 مولّد مراجعة الأدبيات** — موضوع → مسودة Literature Review باستشهادات حقيقية + قائمة مراجع.
- **P3 📊 المساعد الإحصائي (قاتل SPSS)** — بيانات → t-test/ANOVA/انحدار/Cronbach + تفسير عربي + جداول APA.
- **P4 ✍️ جناح المراجعة والتقديم** — رد على المُحكّمين + فحص جاهزية التقديم + تلميع لغوي.

### للطلبة + القيمة المتكررة
- **P5 📬 الموجز الأسبوعي المخصّص** — أوراق/اقتباسات/ترند/مؤتمرات بتقرير أسبوعي. *المحرّك الأهم للاشتراك المتكرر.*
- **P6 🎓 وضع المدرّس (Tutor)** — ناقش أي ورقة/كتاب وافهمها بالعربي بمستواك.
- **P7 🧠 حقيبة الدراسة الذكية** — اختبارات + فلاش كاردز + مراجعة متباعدة.
- **P8 🕸️ رادار البحث** — خريطة استشهادات + لوحة تأثيرك (h-index واقتباساتك + تنبيهات).

### الجولة ٢ — إضافات Premium جديدة
- **P9 ✨ محسّن وإعادة صياغة أكاديمي** — إعادة صياغة بنبرات متعددة (أرصن/أبسط/أوضح/أقصر) + تحسين الإنجليزية الأكاديمية لغير الناطقين + حفظ المعنى والاقتباسات (ع+إ). *بديل QuillBot — يبرّر الاشتراك لوحده. تحرير شريف، بلا عدّاد كسر كاشفات.*
- **P10 🎤 تحضير المناقشة (Viva Prep)** — رفع الأطروحة → هيكل شرائح عرض + **٥٠ سؤال متوقع من اللجنة** بإجابات نموذجية مبنية على نقاط ضعف البحث.
- **P11 📐 مدقّق LaTeX/Word آمن** — يدقّق `.tex`/`.docx` لغوياً **بدون تخريب** المعادلات/الجداول/الأكواد، ويرجّع نفس التنسيق.
- **P12 🔍 فحص التشابه المبدئي** — مقابل المصادر المفتوحة فقط؛ **استرشادي، ليس بديل Turnitin** — يكشف التشابه لتصلّحه (لا يخفيه).
- **P13 🎓 حاسبة الترقيات العلمية (الجدول الأول/الثاني)** — التدريسي يُدخل أنشطته (بحوث/كتب/إشراف/مؤتمرات/براءات) + الرتبة الحالية + تاريخ آخر ترقية/تعيين → البوت يحسب **النقاط** حسب الجداول الرسمية، يبيّن **كم ناقصه** للحد، ويحسب **تاريخ الاستحقاق** للتقديم + قائمة المستمسكات. **حسابي (rule-based) بلا كلفة AI، دقيق وموثوق.** الجداول ومدد الخدمة **قابلة للتعديل من الأدمن** (تتغير بتعليمات الوزارة). يتكامل مع فاحص الـ Quartile لاحتساب نقاط البحث تلقائياً. *استرشادي — القرار النهائي للجنة الترقيات.*

### ترقيات على الموجود
- **P2** → وضع «Deep Research»: خطوات مؤتمتة (بحث → تحليل → كتابة) تسلّم **تقرير Word** منظّم بمراجع حقيقية.
- **P3** → يضيف **توليد رسوم بيانية + كتابة قسم النتائج والمناقشة** أكاديمياً.
- **#20** → ترجمة **سياقية حسب التخصص** (مصطلحات معتمدة لكل مجال).
- **#24** → **تحليل دفعة (batch)** لعدة ملفات PDF دفعة وحدة.
- **P8** → نسخة خفيفة من **سياق الاستشهاد** (داعم/مذكور) عبر Semantic Scholar.
- **Group A** → **تنبيهات التمويل/المنح** ضمن محرك الإعلانات.
- **لاحقاً:** مزامنة سحابية (Google Drive أولاً) — OAuth شغل إضافي.

### 💠 امتيازات Premium (perks، مو ميزات منفصلة)
أولوية المعالجة وقت الذروة · سقوف AI أعلى/غير محدودة · سياق موسّع (ضمن حد DeepSeek ~64K) · تجربة بلا إعلانات.

## 🎛️ قدرات لوحة الأدمن
- نشر/تعديل الإعلانات + الجداول (مجلات، APC، تخصصات).
- مفاتيح الميزات (تشغيل/إيقاف + free/premium + سقوف).
- مفتاح التحصيل العام (monetization on/off).
- منح Premium يدوي (دائم/مؤقت/جماعي) + أكواد هدية.
- بحث/عرض/إلغاء المستخدمين + إحصائيات الدخل والاشتراكات.

## 🗺️ خارطة الطريق
- **مرحلة ٠ — الهيكل:** Go module + config + store + bot core + gate + طبقة AI + بنية الـ flags/billing (نائمة) + لوحة admin.
- **مرحلة ١ — الجمهور (MVP):** المجموعة A + الاقتباس التلقائي (#10). 🟢
- **مرحلة ٢ — نواة المرجع:** بقية B.
- **مرحلة ٣ — المساعد الذكي:** C + P1–P4 + P9 (محسّن الصياغة) + P11 (مدقّق LaTeX) + P13 (حاسبة الترقيات).
- **مرحلة ٤ — المتقدّم:** D + P5–P8 + P10 (تحضير المناقشة) + P12 (فحص التشابه).
- **مرحلة ٥ — التفعيل النقدي:** قلب `monetization_enabled` بعد نمو الجمهور.

> **MVP** يطلق مجاناً (monetization OFF) لبناء الجمهور؛ المدفوع يُبنى نائماً ويُفعّل لاحقاً بضغطة.
